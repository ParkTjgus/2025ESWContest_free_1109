# 스마트 헬스 서포터

![Project Badge](https://img.shields.io/badge/Smart%20Health-Supporter-blue)

**스마트 헬스 서포터**는 저비용 스마트 운동 디바이스 및 안드로이드 앱입니다.  
기존 아날로그 운동기구에 스마트 기능을 결합하여 운동 데이터를 수집, 분석하고 실시간 피드백을 제공합니다.  
현재 급성장 중인 디지털 헬스케어 시장을 겨냥하여, 비용 부담 없이도 체계적인 운동 관리가 가능합니다.

---

## 🏋️‍♂️ 핵심 기능

| 기능                    | 설명                                                                             |
| ----------------------- | -------------------------------------------------------------------------------- |
| **운동 횟수 자동 기록** | 세트 수와 횟수를 앱에 입력하면, 디바이스가 운동 시 자동으로 횟수를 인식하고 기록 |
| **운동 균형 감지**      | 좌우 힘의 불균형을 감지하고 사용자에게 실시간 알림 제공                          |
| **운동 속도 피드백**    | 동작 속도를 분석하여 너무 빠르거나 느린 경우 피드백 제공                         |
| **운동 루틴 관리**      | 날짜별 누적 데이터를 분석해 체계적인 운동 계획 제공                              |
